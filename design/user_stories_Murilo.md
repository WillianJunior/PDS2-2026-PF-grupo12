# User Stories - Receituário Virtual

## US01: Cadastro de Receitas
**Descrição:** Como **Cozinheiro**, quero cadastrar minhas próprias receitas para que ela possa ser consultada digitalmente.
**Critérios de Aceitação:**
* O sistema deve permitir inserir título, lista de ingredientes e modo de preparo.
* O sistema deve permitir definir o tempo de preparo e o nível de dificuldade.
* O sistema deve validar se o campo "título" não está vazio antes de salvar.
* O sistema deve permitir associar uma categoria à receita (ex: Doce, Salgado, Vegano).

## US02: Avaliação de Receitas
**Descrição:** Como **Membro da comunidade**, quero avaliar receitas de outros usuários para destacar as melhores opções.
**Critérios de Aceitação:**
* O sistema deve permitir dar uma nota de 1 a 5 estrelas.
* O sistema deve exibir a média de avaliações na página da receita.
* O sistema deve permitir que o usuário altere sua nota posteriormente.
* O sistema deve exibir o número total de avaliações recebidas.
