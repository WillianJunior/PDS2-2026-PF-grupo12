\## US03: Ferramenta de Pesquisa

\*\*Descrição:\*\* Como \*\*Cozinheiro casual\*\*, quero pesquisar por receitas específicas ou de um grupo específico para reproduzi-las.

\*\*Critérios de Aceitação:\*\*

\* O sistema deve permitir pesquisar e exibir uma receita por título.

\* O sistema deve permitir filtrar receitas por nota.

\* O sistema deve permitir filtrar receitas por dificuldade.





\## US04: Edição e Remoção de Receitas Próprias

\*\*Descrição:\*\* Como \*\*Chef e usuário frequente\*\*, quero atualizar ou remover receitas que já adicionei anteriormente.

\*\*Critérios de Aceitação:\*\*

\* O sistema deve permitir procurar pelas receitas de autoria do próprio usuário.

\* O sistema deve permitir alterar atributos das receitas próprias (título, ingredientes, preparo, etc.).

\* O sistema deve permitir remover por completo uma receita própria.
