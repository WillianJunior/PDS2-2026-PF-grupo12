# User Stories - Receituário Virtual

## US05: "Lista de compras" de ingredientes.
**Descrição:**  Como **Usuário casual** da aplicação, quero ser capaz de marcar quais ingredientes eu possuo para uma
receita, para que os ingredientes que eu não possuo sejam listados para mim.
**Criterios de aceitação:**
* O sistema deve permitir selecionar ingredientes dentre os listados na descrição da receita.
* O sistema deve verificar os ingredientes selecionados contra os ingredientes pedidos na receita.
* O sistema deve mostrar a diferenca entre a lista selecionada e a necessária na forma de uma lista de compras.


## US06: Adicionar favoritos
**Descrição:** Como **Usuário frequente** da aplicação, quero ter a capacidade de separar minhas receitas favoritas num grupo específico, tornando-as mais assessíveis a mim.
**Critérios de aceitação:**
* O sistema deve permitir adição de certa receita a um grupo de receitas "favoritas", separado da maioria.
* O sistema deve permitir a exibição da lista de receitas favoritas do usuário.
* O sistema deve permitir a remoção de receitas da lista de favoritos do usuário.
